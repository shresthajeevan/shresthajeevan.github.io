
# 💼 Personal Portfolio - Jeevan Shrestha

This is my personal portfolio website built using **HTML**, **CSS**, and **Bootstrap 5**. It showcases my experience, skills, and selected projects. The design is fully responsive and includes smooth animations powered by the **AOS (Animate On Scroll)** library.

## 🌐 Live Preview
[Portfolio Website](#) *(https://shresthajeevan.github.io/)*

---

## 📂 Features

- Responsive layout using Bootstrap  
- Clean, modern design with custom styling  
- Smooth animations with AOS library  
- Project portfolio section with tags and live/demo links  
- Downloadable resume  
- Animated introduction and skills overview  

---

## 🚀 Technologies Used

- **HTML5**  
- **CSS3**  
- **Bootstrap 5**  
- **JavaScript**  
- **AOS (Animate on Scroll)**  
- **Google Fonts**  
- **Bootstrap Icons**
